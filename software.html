Instalación de R y R-studio

Windows

  1. Ve al sitio web de R en https://cran.r-project.org/.
  2. Haz clic en Download R for Windows.
  3. Haz clic en base y luego en Download R-4.3.2 for Windows para descargar el instalador
  4. Ejecuta el instalador base y sigue las instrucciones que aparecen para completar la instalación de R.
  5. A continuación, ve al sitio web de R-Studio en https://www.rstudio.com/products/rstudio/download/#download.
  6. Descarga el instalador de R-Studio correspondiente a Windows.
  7. Ejecuta el instalador de R-Studio y sigue las instrucciones para completar la instalación.
  8. Una vez instalados, puedes abrir R-Studio y comenzar a trabajar con R.

macOS

  1. Ve al sitio web de R en https://cran.r-project.org/.
  2. Haz clic en Download R for macOS..
  3. Descarga el archivo .pkg y ejecútalo.
  4. Sigue las instrucciones del instalador para completar la instalación de R.
  5. A continuación, ve al sitio web de R-Studio en https://www.rstudio.com/products/rstudio/download/#download.
  6. Descarga el instalador de R-Studio correspondiente a macOS.
  7. Ejecuta el instalador de R-Studio y sigue las instrucciones para completar la instalación.
  8. Una vez instalados, puedes abrir R-Studio y comenzar a trabajar con R.

Instalación de Python y Spyder

Windows

Python

  1. Descarga el instalador de Python desde python.org.
  2. Ejecuta el instalador y sigue las instrucciones en pantalla.
  3. Asegúrate de marcar la opción "Add Python X.X to PATH" durante la instalación.
  4. Verifica la instalación abriendo la línea de comandos y escribiendo python --version.

Spyder

  1. Abre la línea de comandos.
  2. Escribe el siguiente comando y presiona Enter:
    3. pip install spyder
  3. Inicia Spyder desde el menú de inicio o ejecutando spyder desde la línea de comandos.

macOS

Python

  1. Python generalmente ya está preinstalado en macOS. Si deseas instalar una versión específica, puedes hacerlo desde python.org.

Spyder

  1. Abre la terminal.
  2. Escribe el siguiente comando y presiona Enter:
    3. pip install spyder
  3. Inicia Spyder desde Launchpad o ejecutando spyder desde la terminal.
